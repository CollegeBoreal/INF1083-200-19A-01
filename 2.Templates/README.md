# TNS CLI Blank Template

### :one: Créer un projet, remplacer `my` dans `my-blank-ng` par b`<`votre :id:`>`

:bulb: par example le nom du projet sera `b300098957-blank-ng` 

* Création

```
$  tns create my-blank-ng --template tns-template-blank-ng
```

* Émulation

```
$  tns run
```
